Student Portal

A responsive Student Portal developed using Flask for the backend and HTML, CSS, and JavaScript for the frontend. This project is designed to provide a simple and user-friendly platform where students can access academic information and manage their activities.

Features
Clean and responsive user interface
Modern navigation bar
Flask template rendering with Jinja2
Static asset management (CSS and images)
Modular project structure
Easy to extend with new features
Technologies Used
Python
Flask
HTML5
CSS3
JavaScript
Jinja2 Templates
Future Improvements
User authentication (Login & Registration)
Student Dashboard
Course Registration
Grade Management

Profile Management
